iPXE based bootserver

This application is designed to ease the installation of debian operating system by automating the bulk of the interaction in the installation process.

It leverages the excellent PXE boot environment iPXE using a Flask based python application to provide ipxe menus, debian-installer scripts and post install scripts.

Depending on system config and your choice of image it also uses Salt stack to provide configuration management and orchestration of systems.

The basic premise of the boot server is to provide a simple and reliable way to deploy and configure linux servers with a minimum of mess and fuss. The aim is to be able to deploy, redeploy and build complex linux systems in hours rather than days.
